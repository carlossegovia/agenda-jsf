# Programación Web Front-End || JSF
CRUD Simple en JSF para la materia Programación Web Frontend de la Facultad Politécnica de la Universidad nacional de Asunción. 

## Herramientas y librerías utilizadas

> * Maven
> * Eclipse
> * JBOSS
> * Java 7
> * Primefaces 6.0
> * Gson
